# 海光平台实现 Double Blocking QR 分解

## 编译
执行以下命令编译程序：
```bash
make
```

## 执行
运行程序时使用以下命令：
```bash
./mydoubleblocking_Iter_V6 m n nb b
```

### 参数说明
- `nb` 必须大于 `b`：`nb > b`
- 建议设置：`b = 32`
